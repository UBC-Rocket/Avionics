TO MAKE A BIN FOLDER FOR THE MPL AND MPU INPUT DATA:

1. export the .xlsx data file to .csv
2. change the Matlab script to target the named file
3. empty the testMPL.bin and testMPU.bin files (delete all of the stuff in them or make new ones)
4. run the matlab script
5. replace the files on the sd card with these new bin files
6. change the sensor files in the state mecheine to use the sim.h headers. 4 location to change.
7. run that shit.
